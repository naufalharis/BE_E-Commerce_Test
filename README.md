Tech Stack
-------------
Node.js + Express

PostgreSQL

Sequelize ORM

JWT Authentication

Midtrans Snap (Sandbox)

Postman (API Testing)

Install Dependencies:
---------------------
```
npm install express sequelize pg pg-hstore dotenv midtrans-client
```
```
npm install --save-dev nodemon
```

Install & Setup Midtrans Snap:
```
npm install midtrans-client
```
